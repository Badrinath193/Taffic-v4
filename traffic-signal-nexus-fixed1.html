// MODULE_03
System Architecture

7-layer modular middleware with full bidirectional V2X communication, real OSM data, and Unity3D visualization bridge.

LAYER 01
OSM Ingestion
Overpass API. Only real highway=traffic_signals. Bidirectional edge splitting. Junction segmentation at shared nodes.
LAYER 02
Vehicle Engine
2000-vehicle capacity. Bidirectional pathfinding. 5 types: Car, Truck, Bus, Motorcycle, Emergency. Physics-based stopping.
LAYER 03
MARL Policy
MA2C + RESCO + RouteRL. Per-intersection RL agents. Cooperative reward via queue-length minimization.
LAYER 04
V2X Protocol
Vehicle-to-Infrastructure messages: QUEUE_UPDATE, ROUTE_INTENTION, PHASE_CHANGE, EMERGENCY preemption.
LAYER 05
Signal Control
Adaptive cycle durations. Phase optimization per RL agent. Cross-intersection coordination. Emergency override.
LAYER 06
Unity3D Bridge
TCP co-simulation. Google Maps tiles. Per-vehicle 3D models. 5 camera modes. Day/night cycle. 60Hz sync.
LAYER 07
Analytics
Throughput, queue, emissions, safety. Rolling window charts. Fleet status grid. Export-ready telemetry.
